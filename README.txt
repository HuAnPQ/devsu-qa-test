A continuación se detalla las instrucciones de ejecución del proyecto java con Gradle para el ejercicio de API DemoBlaze y el proyecto de Playwright para el ejercicio E2E:

1. Clonar el repositorio

2. Abrir el archivo de prueba login.feature o signup.feature.

3. Ejecutar la prueba del escenario, en caso de playwright usar el comando npx playwright test.

4. Validar los resultados del escenario y sus pasos.
